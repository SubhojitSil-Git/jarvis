# 🧠 Jarvis (Learning Project)

A simple AI assistant built as part of my exploration into Python, automation, and basic AI concepts.

---

# 📌 What This Is

This project is a **learning-focused implementation** of a Jarvis-like assistant.  
The goal was to understand how AI assistants work at a basic level — not to build a production-ready system.

---

# ⚙️ What It Can Do

- Take basic user input (voice/text depending on setup)  
- Perform simple automated tasks  
- Fetch information from the web  
- Execute predefined commands  

---

# 🧩 Tech Used

- Python  
- Basic NLP / command handling  
- APIs (where applicable)  
- System automation libraries  

---

# 🚧 Limitations

- Not a real AI (no advanced intelligence)  
- Limited command handling  
- No deep learning or custom model  
- Not optimized for real-world use
